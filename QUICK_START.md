# ⚡ 快速部署 - 3分钟搞定

## 📦 你需要做的事

只需要 **3 个步骤**！

---

## 步骤 1️⃣：替换文件

1. 下载新的 `index.html`
2. 复制到项目文件夹，覆盖旧文件
3. 完成！

---

## 步骤 2️⃣：提交到 GitHub

打开命令行/终端，执行：

```bash
# Windows 用户
cd C:\Users\Henry Peng\Desktop\word-project\Life\travel-calendar-project

# Mac/Linux 用户
cd ~/path/to/travel-calendar-project

# 执行以下命令
git add index.html
git commit -m "优化导出功能 + 移动端详情弹窗"
git push
```

---

## 步骤 3️⃣：等待部署

1. 推送后等待 **1-2分钟**
2. 访问你的网站
3. 按 `Ctrl + Shift + R` 强制刷新
4. 完成！✨

---

## ✅ 快速测试

### 测试导出功能：
1. 添加一个测试旅行
2. 点击"📤 导出"
3. 打开下载的文件
4. 确认新旅行在文件中

### 测试移动端弹窗（手机上）：
1. 点击已有旅行日期
2. 应该弹出详情卡片
3. 底部有"✏️编辑"和"关闭"按钮

---

## 🐛 出问题了？

- 页面没变化？→ 强制刷新（Ctrl + Shift + R）
- 部署失败？→ 查看 Render 的 Logs
- 其他问题？→ 查看 `DEPLOYMENT_GUIDE.md` 详细文档
